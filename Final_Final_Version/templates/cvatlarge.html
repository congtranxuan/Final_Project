Computer Vision At Large


In this section we can see a collection of companies that supply, use or create machine vision software for facial and object recognition, machine learning and AI. Some of the ways that Facial Recognition software is being used today are by law enforcement and security agencies, casinos, dating sites, social media to tag people automatically, upscale hotels to greet guests on arrival, bars and restaurants to pick out underage drinkers, retail stores that do real-time sentiment analysis to predict buying behavior.
